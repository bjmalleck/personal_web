Projects

Mine488 - Visit Page

Mine488 is a basic interactive voxel engine written in Javascript using WebGL/Canvas. It was developed during a two week time frame as my final project for the Introduction to Computer Graphics (CSS 488) course at the University of Waterloo in Spring 2015. It includes movement and camera view, a basic physics engine with limited collision, gravity and friction, as well as basic procedural world generation.
